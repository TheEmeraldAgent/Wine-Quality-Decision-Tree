# Wine-Quality-Decision-Tree

Machine learning project predicting wine quality using decision tree models with exploratory data analysis and feature importance evaluation in Python.

## Project Overview

This project analyzes the relationship between physicochemical properties of wine and quality scores.  
A decision tree model is used to predict wine quality and identify the most important variables affecting the outcome.

The notebook includes data preprocessing, exploratory analysis, model training, and evaluation.

## Dataset

The dataset contains physicochemical measurements of wine samples including:

- fixed acidity
- volatile acidity
- citric acid
- residual sugar
- chlorides
- free sulfur dioxide
- total sulfur dioxide
- density
- pH
- sulphates
- alcohol

These variables are used to predict the **wine quality score**.

## Technologies Used

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn  
Jupyter Notebook

## Machine Learning Model

A Decision Tree model is used to analyze the dataset and predict wine quality.  
The model also helps identify which chemical attributes most strongly influence wine quality.
